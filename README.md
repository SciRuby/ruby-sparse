# Ruby-Sparse

Ruby Library for Sparse Tensors.

## Installation

### Build the library

```sh
git clone https://github.com/sciruby/ruby-sparse
cd ruby-sparse/
gem install bundler
bundle install
rake compile
```

### Run tests

```sh
rake test
```

### Try out the code without installing

```sh
rake pry
```
